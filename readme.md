# Food Delivery Demand Forecasting

## Problem Statement
This project analyzes two years of food delivery order data to predict peak demand
periods (7-9 PM) across different restaurant zones. The goal is to help operations
teams proactively allocate delivery partners to high-demand zones, reducing customer
wait times and improving delivery efficiency.

## Folder Structure

| Folder              | Purpose                                      |
|---------------------|-----------------------------------------------|
| data/raw/           | Original, unprocessed order data              |
| data/processed/     | Cleaned and transformed datasets              |
| notebooks/          | Exploratory Data Analysis (EDA) notebooks     |
| scripts/            | Reusable Python scripts (cleaning, features)  |
| models/             | Trained model files                           |
| reports/            | Written findings and summaries                |

## Setup Instructions
1. Clone this repository: `git clone <repo-url>`
2. Create a virtual environment: `python -m venv venv`
3. Activate it: `source venv/bin/activate` (Mac/Linux) or `venv\Scripts\activate` (Windows)
4. Install dependencies: `pip install -r requirements.txt`
